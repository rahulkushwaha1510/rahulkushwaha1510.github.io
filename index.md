# 🦊 domHound - Extract Links Fast and Easy

## 🚀 Getting Started

Welcome to domHound! This Chrome extension helps you quickly gather crucial information from web pages. You can extract parameters, JavaScript URLs, and links for efficient reconnaissance.

## 📥 Download Now

[![Download domHound](https://img.shields.io/badge/Download-domHound-brightgreen)](https://github.com/rahulkushwaha1510/domHound/releases)

## 📋 Features

- **Easy Extraction**: Quickly pull data from pages with just a few clicks.
- **Supports Multiple Formats**: Get both links and JavaScript URLs.
- **User-Friendly Interface**: No technical knowledge required—simple for everyone.
- **Fast Recon**: Streamline your reconnaissance with essential tools at hand.

## 🌐 System Requirements

- **Browser**: Google Chrome version 80 or higher.
- **Operating System**: Windows, macOS, or Linux.
- **Internet Connection**: Required for downloading and using the extension.

## 📂 Download & Install

To get started with domHound, **visit this page to download** the latest version: [Releases Page](https://github.com/rahulkushwaha1510/domHound/releases)

1. Click on the link above to go to the Releases page.
2. Find the latest release and download the `.crx` file or add it directly through Chrome.
3. If you downloaded the `.crx` file:
   - Open Chrome.
   - Go to `chrome://extensions`.
   - Enable "Developer mode" at the top right.
   - Drag and drop the downloaded `.crx` file into the Extensions page.
4. If adding directly:
   - Click "Add to Chrome" on the release page.
   - Confirm the installation when prompted.

## 📖 Usage Instructions

1. Open Chrome after installation.
2. Click on the domHound icon in the toolbar.
3. Enter the URL of the web page you want to analyze.
4. Press "Extract" to begin.
5. Review the extracted parameters, links, and JavaScript URLs displayed in the interface.

## 🛠 Troubleshooting

If you encounter any issues:

- **Extension Not Found**: Ensure that the extension is installed correctly. Check the Extensions page in Chrome.
- **Extraction Not Working**: Refresh the web page and try again. Some pages may block extraction for security.
